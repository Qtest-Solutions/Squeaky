================================================================================
  CUSTOMIZATION GUIDE
================================================================================

UPDATE PAGE CONTENT:
1. Open app/[page]/page.tsx
2. Edit the JSX content directly
3. Save file (hot reload in dev server)

UPDATE NAVIGATION:
1. Edit <nav> sections in each page
2. Update Link href attributes
3. Change active page styling

ADD NEW PAGE:
1. Create folder: app/newpage/
2. Create file: app/newpage/page.tsx
3. Copy from existing page as template
4. Add navigation link to other pages
